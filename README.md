# Baky Browser Landing Page

Landing page pre `Baky Browser`, produkt firmy `Baky OS`.

## Lokálne spustenie

1. Nainštaluj balíčky:
   `npm install`
2. Spusti vývojový server:
   `npm run dev`
3. Build:
   `npm run build`

## Čo je upravené

- odstránené rozbité slovenské texty
- odstránené fake štatistiky a prehnané marketingové tvrdenia
- odstránené generické AI Studio texty
- zjednotený obsah na reálny produkt `Baky Browser` od `Baky OS`

## Vercel

Tento projekt je pripravený ako statická `Vite` stránka pre `Vercel`.

- framework: `Vite`
- build command: `npm run build`
- output directory: `dist`

Stačí nahrať projekt na GitHub a importnúť ho do Vercelu.
